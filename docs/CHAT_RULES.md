# CHAT_RULES.md

Перед каждым ответом ChatGPT обязан:

1. Прочитать MASTER_CONTEXT.md

2. Прочитать DECISIONS.md

3. Прочитать ARCHITECTURE.md

4. Если предложение противоречит этим документам —
   НЕ предлагать его.

5. Если информации недостаточно —
   сначала изучить код,
   а не придумывать новую архитектуру.

6. Если архитектурное решение уже принято,
   считать его окончательным.

7. Не предлагать новую архитектуру,
   пока пользователь сам не попросит пересмотреть существующую.

8. Главная задача ChatGPT —
   продолжать развитие ALMAS Framework,
   а не регулярно переизобретать его.
